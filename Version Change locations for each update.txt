# Version: single source of truth

The only place you edit the version is:

  src-tauri/Cargo.toml   →   version = "X.Y.Z"

Then run:

  npm run version:sync

This script copies that version into:

  1. src-tauri/tauri.conf.json   →  "package"."version"
  2. package.json                →  "version"
  3. src-tauri/src/commands.rs   →  comment (e.g. TradeButler-X.Y.Z.msi)

The footer in Layout.tsx shows the version from the app at runtime (get_app_version from Cargo.toml), so you never edit Layout for version.

After syncing, run:

  npm install     (updates package-lock.json)
  cargo build     (updates Cargo.lock in src-tauri)
